Projektin yleiskatsaus

STUNITE on viiden tietojenkäsittelyopiskelijan Ticorporate-projekti syksyltä 2024. Sovellus on suunnattu tapahtumakävijöille ja -järjestäjille, tavoitteena tuoda opiskelijatapahtumat laajemmin esille ja edistää eri järjestöjen välistä yhteistyötä.

Projektissa roolit jakautuivat Backend- ja Frontend-kehittäjään, pilvipalveluosaajaan, UI/UX-suunnittelija & sisällöntuottajaan sekä liiketoimintaan keskittyvään osaajaan. Oma pääroolini oli Frontend-kehittäjä, ja sivuroolissani toimin testaajana. Sovellus ei ole tällä hetkellä tuotantokäytössä

Tapahtumahaku ja suodatus

updateFilteredEvents() päivittää suodatetut tapahtumat käyttäjän hakutermien ja valintojen perusteella. Funktio tukee hakua nimen, kaupungin, tagien ja päivämäärävälin perusteella, ja lajittelee tulokset aikajärjestykseen.

Reaaliaikainen haku Monikriittinen suodatus Tulosten järjestäminen
event-filter.service.ts 
updateFilteredEvents(eventsData: Event[]) {
  this.filteredEventData = computed(() => {
    const search = this.searchTerm().toLowerCase();
    const city = this.selectedCity();
    const tag = this.selectedTag();
    const { start, end } = this.selectedDateRange();

    let filteredEvents = eventsData.filter((event) => {
      const matchesSearch = event.eventName.toLowerCase().includes(search);
      const matchesCity = city ? event.city === city : true;
      const matchesTag = tag ? event.eventTags?.includes(tag) : true;

      let matchesDate = true;
      if (start || end) {
        const eventStartDate = this.parseCustomDate(event.date);
        matchesDate =
          (!start || eventStartDate >= new Date(start)) &&
          (!end || eventStartDate <= new Date(end));
      }

      const currentDate = new Date();
      currentDate.setHours(0, 0, 0, 0);

      const eventEndDate = event.endingDate
        ? this.parseCustomDate(event.endingDate)
        : this.parseCustomDate(event.date);
      eventEndDate.setHours(0, 0, 0, 0);

      const isFutureEvent = eventEndDate >= currentDate;

      return (
        matchesSearch &&
        matchesCity &&
        matchesTag &&
        matchesDate &&
        isFutureEvent
      );
    });

    filteredEvents = filteredEvents.sort((a, b) => {
      const dateA = this.parseCustomDate(a.date);
      const dateB = this.parseCustomDate(b.date);
      return dateA.getTime() - dateB.getTime();
    });

    return filteredEvents;
  });
}

Järjestäjäkalenteri

eventsUpdated() hallinnoi järjestäjäkalenterin tapahtumien näyttämistä ja suodatusta. Funktio jakaa tapahtumat tuleviin ja menneisiin, ja päivittää kalenterinäkymän reaaliajassa käyttäjän valintojen mukaan.

Kalenterin integraatio Dynaaminen suodatus
organizer-calendar.component.ts 
eventsUpdated() {
  const eventsData = this.events();
  const currentDate = new Date();

  let filteredEvents = eventsData;

  // Suodata kaupungin mukaan
  if (this.selectedCity) {
    filteredEvents = filteredEvents.filter(
      (event) => event.city.toLowerCase() === this.selectedCity.toLowerCase()
    );
  }

  // Suodata tagien mukaan
  if (this.selectedTags && this.selectedTags.length > 0) {
    filteredEvents = filteredEvents.filter((event) =>
      event.eventTags.includes(this.selectedTags)
    );
  }

  // Suodata järjestäjän mukaan
  if (this.selectedOrganizer) {
    filteredEvents = filteredEvents.filter((event) => {
      return (
        event.organizationName.toLowerCase() ===
        this.selectedOrganizer.toLowerCase()
      );
    });
  }

  // Jaa tuleviin ja menneisiin tapahtumiin
  this.upcomingEvents = filteredEvents
    .filter((event) => {
      const eventDate = new Date(this.formatDateToISO(event.date));
      return eventDate >= currentDate;
    })
    .sort((a, b) => {
      const dateA = new Date(this.formatDateToISO(a.date) + 'T' + a.startingTime);
      const dateB = new Date(this.formatDateToISO(b.date) + 'T' + b.startingTime);
      return dateA.getTime() - dateB.getTime();
    });

  // Päivitä kalenterinäkymä
  if (filteredEvents.length > 0) {
    this.calendarOptions.events = filteredEvents.map((event) => ({
      title: event.eventName,
      date: this.formatDateToISO(event.date),
      color: event.status === 'Varattu' ? '#fe7775' : '#f0d37c',
      url: `/events/${event._id}`,
      extendedProps: {
        startingTime: event.startingTime,
        venue: event.venue,
        organizationName: event.organizationName,
        eventId: event._id,
      },
    }));
  }
}

Tapahtuman statushallinta

getPublishedEvents() suodattaa tapahtumat niin, että vain vahvistetut ja julkaistut tapahtumat näkyvät käyttäjille. Järjestäjäkalenterissa värikoodaus auttaa erottamaan alustavat ja vahvistetut tapahtumat.

Julkaisun hallinta Visuaalinen erottelu
event.service.ts 
getPublishedEvents(): Observable {
  return this.http.get(this.apiUrl).pipe(
    map((events) =>
      events.filter((event) => {
        const now = new Date();
        const publishDate = new Date(event.publishDateTime);

        return (
          event.status === 'Varattu' &&
          event.publishDateTime != null &&
          publishDate <= now
        );
      })
    )
  );
}

End-to-End testaus Cypressillä

E2E-testi, joka testaa koko tapahtuman elinkaaren: järjestäjän sisäänkirjautumisen, tapahtuman luomisen, sen näkyvyyden eri näkymissä ja lopulta poistamisen. Testi varmistaa kriittisten toimintojen toimivuuden.

Cypress E2E Testaus
create-event.cy.ts 
describe('Create Event', () => {
  beforeEach(() => {
    cy.visit('http://localhost:4200/');
    
    // Avaa kirjautumismodaali
    cy.get('.login-icon > .material-symbols-outlined')
      .should('be.visible')
      .click();

    // Siirry järjestäjän kirjautumiseen
    cy.get('.login-buttons > :nth-child(2)').click();

    // Kirjaudu sisään
    const email = Cypress.env('CYPRESS_ORGANIZER_EMAIL');
    const password = Cypress.env('CYPRESS_ORGANIZER_PASSWORD');
    
    cy.get('#email').type(email);
    cy.get('#password').type(password);
    cy.get('form.ng-dirty > button').click();

    // Varmista onnistunut kirjautuminen
    cy.url().should('include', '/organizer-view');
  });
  
  it('luo tapahtuma ja varmista sen näkyvyys', () => {
    // Siirry tapahtuman luomiseen
    cy.get('.organizer-buttons > :nth-child(1)').click();

    // Täytä lomake
    cy.get('input[name="eventName"]').type('TestiTapahtumaCypress');
    cy.get('[name="eventDate"]').type(formattedDate);
    cy.get('[name="eventTime"]').type('12:00');
    cy.get('input[name="city"]').type('Helsinki');
    cy.get('input[name="venue"]').type('Messukeskus');
    cy.get('textarea[name="details"]').type('Testitapahtuman kuvaus.');
    
    // Lähetä lomake
    cy.get('form.create-event-form').submit();

    // Varmista onnistunut luominen
    cy.get('.ng-trigger')
      .should('contain', 'Tapahtuman luonti onnistui.');

    // Tarkista näkyvyys järjestäjän näkymässä
    cy.get('.page-container > :nth-child(3)')
      .should('contain', 'TestiTapahtumaCypress');
  });

  it('poista tapahtuma ja varmista poistuminen', () => {
    cy.get('.your-events')
      .contains('.event-name', 'TestiTapahtumaCypress')
      .parents('.your-events')
      .within(() => {
        cy.contains('Poista').click();
      });

    cy.on('window:confirm', () => true);
    
    cy.get('.events-div')
      .should('not.contain', 'TestiTapahtumaCypress');
  });
});
