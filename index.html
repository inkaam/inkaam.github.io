Tietoa minusta

Olen JAMKin tietojenkäsittelyn opiskelija opintojeni loppuvaiheessa. Tällä hetkellä haen harjoittelupaikkaa ja täydentelen samalla opintojani Fullstack Open MOOC -kurssilla.

Vahvinta osaamistani ovat Angular ja TypeScript sekä käyttöliittymien rakentaminen. Frontend on se missä viihdyn parhaiten, mutta minulla on myös kokemusta backend-puolelta (Node.js, REST API -toteutukset).

Haluan jatkuvasti oppia uutta ja kehittää osaamistani monipuolisesti ohjelmistokehityksen eri osa-alueilla. Olen valmis opettelemaan tarvittavat taidot harjoittelua ja tulevia työtehtäviä varten.

Osaaminen

Frontend: Angular, TypeScript, HTML, CSS

Backend (perusteet): Node.js, REST API

Työkalut: Git, VS Code, Postman, Figma

Muu: ohjelmistotestauksen perusteet, ryhmätyöskentely, projektinhallinta
